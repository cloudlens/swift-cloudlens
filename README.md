# CloudLens Module for Swift

## Build

Build with

`swift build --config release`

## Example

Run example with

`.build/release/Main`

## Run-Eval-Print Loop

Invoke read-eval-print loop with

`swift -I.build/release -L.build/release -lCloudLens`

Import CloudLens module with

`import CloudLens`

## Xcode workspace

`open CloudLens.xcworkspace`
